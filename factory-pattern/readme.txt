Factory:
    - A building where things are manufactured
    - In programming world, a factory is just an object that creates or manufactures different objects.

    - factories allow us to handle all of our object creation in a centralized location.
    - we dont need to worry object creations using new keyword all over our code.
    - clean and helps keep our code concise.

- clean implementation of creating many versions of many different types of object.

Scenario 1:
----------
- I own a software business. I own employees
- 2 types of employees 
    - developers 
    - testers
- I want to setup a database to keep track of my employees
- I need a clean and concise way of creating new employees and inserting them to my database.
